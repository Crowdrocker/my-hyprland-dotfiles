# My Hyprland Dotfiles

Custom Hyprland setup based on ML4W dotfiles with personal tweaks.

## Features
- Gaming mode toggle (Super + F3)
- Optimized for AMD GPU
- Custom keybinds and workflow

## Installation
1. Install Arch Linux
2. Run Chris Titus Linutil
3. Install ML4W dotfiles
4. Clone this repo and copy configs over

## Hardware
- i7-4790
- RX 580
- 16GB RAM
